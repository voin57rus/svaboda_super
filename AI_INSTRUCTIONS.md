# 🤖 AI-ассистент — инструкция для администратора

## Настройка AI API ключа

1. Зайди в админку → ⚙️ Настройки бота → 🤖 AI ключ
2. Жми "🔑 Вставить API ключ"
3. Отправь свой OpenRouter API ключ (формат: `sk-or-...`)
4. Получить ключ: https://openrouter.ai/keys

## Выдача AI-ключа пользователю

### Способ 1 — Через кнопки (удобно)
1. Админка → ✏️ Изменить тексты → 🤖 AI-доступ
2. Жми "🔑 Выдать AI-ключ"
3. Введи telegram_id или @username пользователя
4. Выбери тариф: **S** (10К), **P** (20К), **V** (50К)
5. Бот сгенерирует ключ → отправь его пользователю

### Способ 2 — Командой
```
/gen_ai_key S          → ключ с 10К токенов
/gen_ai_key P          → ключ с 20К токенов
/gen_ai_key V          → ключ с 50К токенов
/gen_ai_key S 2223     → ключ с 10К + свой ID для отслеживания
```

## Пополнение токенов

### Способ 1 — Через кнопки
1. Админка → ✏️ Изменить тексты → 🤖 AI-доступ
2. Жми "💰 Пополнить токены"
3. Введи telegram_id или @username
4. Введи количество

### Способ 2 — Командой
```
/add_ai_tokens 5191406344 S 1000    → +1000 токенов
/add_ai_tokens 5191406344 P 5000    → +5000 токенов
```

## Формат ключей

Ключ выглядит: `AI-S-A1B2C3D4E5F6`
- **AI** — префикс
- **S/P/V** — тариф
- **A1B2C3D4E5F6** — уникальный ID ключа

## Тарифы

| Буква | Название | Токенов |
|-------|----------|---------|
| **S** | Standard | 10,000 |
| **P** | Premium | 20,000 |
| **V** | VIP | 50,000 |

## Активация пользователем

Пользователь активирует ключ командой:
```
/ai_key AI-S-A1B2C3D4E5F6
```
